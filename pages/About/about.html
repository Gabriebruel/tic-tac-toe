Sobre o Projeto

Este projeto é um Jogo da Velha desenvolvido utilizando HTML, CSS e JavaScript.

Ele conta com modo multiplayer local e também um modo contra inteligência artificial, com diferentes níveis de dificuldade.

O foco principal foi aplicar conceitos de lógica de programação, manipulação do DOM e design de interfaces modernas.
